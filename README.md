# fullpagejs
-------------------------------------------------------------------------------------------------------------------------------
	1.	fullstate()  返回全屏状态 
					 返回值： true  false

	2.	fullElm()    返回处于全屏模式的元素
	                 返回值： 元素对象

	3.	fullscreenchange(fn)监测全屏变化事件 
							fn：事件处理程序

	4.	fullscreen(elm)  进入全屏模式   
						 elm： 指定元素

	5.	exitfullscreen() 退出全屏模式
